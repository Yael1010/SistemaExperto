Hecho para aprender ML

Cómo usar Mini-Weka Pro

  • Suba un archivo CSV con sus datos de entrenamiento
  • La última columna se considera automáticamente como variable objetivo
  • Asegúrese de que no haya valores faltantes críticos

  • ID3: Mejor para datos categóricos y reglas interpretables
  • K-NN: Mejor para patrones complejos y datos numéricos
  • Hold-out: Más rápido, divide 70/30
  • K-Fold: Más robusto, usa validación cruzada

  • Individual: Complete el formulario con un ejemplo
  • Por lotes: Suba CSV con múltiples ejemplos
  • Use las mismas columnas del entrenamiento (excepto la variable objetivo)
