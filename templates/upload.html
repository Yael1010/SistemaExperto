Requisitos del Dataset
  • Formato: Archivo CSV (.csv)
  • Encabezados: Primera fila con nombres de columnas
  • Variable objetivo: Última columna
  • Datos: Sin filas completamente vacías
  • Codificación: UTF-8 recomendado
Consejos
  • Datos faltantes: El sistema puede manejar algunos valores faltantes automáticamente
  • Variables categóricas: Use texto descriptivo (ej: "Sí", "No", "Alto", "Medio", "Bajo")
  • Variables numéricas: Use números decimales con punto como separador
  • Tamaño: Mínimo 20 filas recomendado para entrenamiento efectivo
